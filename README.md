# מערכת הזמנות בשר - Frontend

## התקנה
```bash
npm install
```

## הרצה מקומית
```bash
npm start
```

## הגדרת כתובת השרת
ערוך את `src/config.js` ושנה את `YOUR-BACKEND-NAME` לכתובת השרת האמיתית.

## פריסה ב-Vercel
1. העלה ל-GitHub
2. חבר ל-Vercel
3. פרוס

## תכונות
- חיפוש לקוחות
- עדכון משקלים
